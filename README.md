# jordan-brand-front

## Build Setup

```bash
# install dependencies
$ npm install

# serve with hot reload at localhost:3000
$ npm run dev

# build for production and launch server
$ npm run build
$ npm run start
```


Создайте файл ```.env``` c настройками:
```bash
HOST_PORT=3001
HOST_URL="localhost"
API_URL="http://localhost:3000/api"
HOST_NAME="Jordan Brand :: интернет-магазин"
```
